# 객체지향 콘솔 RPG 프로젝트

C# 객체지향 프로그래밍을 활용하여 제작한 콘솔 기반 턴제 RPG 게임입니다.  
플레이어는 몬스터를 처치하며 성장하고, 장비와 아이템을 활용해 보스 몬스터를 공략할 수 있습니다.

---

# 프로젝트 소개

이 프로젝트는 객체지향 프로그래밍의 핵심 개념인
- 클래스(Class)
- 상속(Inheritance)
- 추상 클래스(Abstract)
- 인터페이스(Interface)
- 다형성(Polymorphism)

등을 활용하여 제작하였습니다.

플레이어는 몬스터와 전투를 진행하며 경험치와 골드를 획득하고,  
장비를 장착하거나 포션을 사용하여 캐릭터를 성장시킬 수 있습니다.

# 클래스 구조

## Character (추상 클래스)

모든 캐릭터의 부모 클래스입니다.

### 주요 기능
- 이름
- 체력
- 공격력
- 공격 기능
- 상태 이상 처리
- 사망 확인

## Player 클래스

Character를 상속받는 플레이어 클래스입니다.

### 주요 기능
- 레벨 시스템
- 경험치 획득
- 레벨업
- 골드 보유
- 장비 장착
- 인벤토리 관리

## Monster 클래스

Character를 상속받는 몬스터 클래스입니다.

### 주요 기능
- 경험치 보상
- 골드 보상
- 보스 몬스터 여부

## Item 클래스

모든 아이템의 부모 클래스입니다.

### 파생 클래스
- Weapon
- Armor
- Potion

## Weapon 클래스

무기 아이템 클래스입니다.

### 기능
- 공격력 증가

## Armor 클래스

방어구 아이템 클래스입니다.

### 기능
- 체력 증가

## Potion 클래스

포션 아이템 클래스입니다.

### 기능
- 체력 회복
- IUsable 인터페이스 구현

## IUsable 인터페이스

사용 가능한 아이템 인터페이스입니다.

csharp
void Use(Player player);


포션 사용 기능을 구현하기 위해 사용했습니다.

## BattleManager 클래스

전투를 관리하는 클래스입니다.

### 주요 기능
- 전투 진행
- 몬스터 공격
- 플레이어 공격
- 포션 사용
- 아이템 드랍

## GameManager 클래스

게임 전체 흐름을 담당하는 클래스입니다.

### 주요 기능
- 게임 시작
- 지역 이동
- 상점
- 인벤토리
- 상태 출력

# 주요 구현 코드

## 추상 클래스 사용

```csharp
abstract class Character
{
    public string Name;
    public int Hp;
    public int Attack;
}
```

캐릭터 공통 기능을 관리하기 위해 사용했습니다.

---

## 인터페이스 사용

csharp
interface IUsable
{
    void Use(Player player);
}
사용 가능한 아이템 기능을 구현하기 위해 사용했습니다.

## 상속 사용
csharp
class Player : Character
csharp
class Monster : Character

Character 클래스를 상속받아 플레이어와 몬스터를 구현했습니다.

## 다형성 사용

csharp
IUsable item = player.Inventory[select] as IUsable;

인터페이스를 활용하여 다양한 사용 아이템을 처리했습니다.
